<h1>Design-A-Date</h1>
Design-A-Date is a tool designed to plan a good time with ease and less indecisiveness. 
This application will populate the nearest cinemas and restaurants by zip code. 

<h1>User Story</h2>


<code>AS a date seeker</code>

<code>I WANT to plan my evenings with convenience</code> 

<code>SO there is less friction in making plans</code>

<h1>API's Used</h1>
<li>MovieDB</li>
<li>GraceNote</li>
<li>Zomato</li>
